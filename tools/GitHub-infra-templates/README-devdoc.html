# Rackspace Cloud Identity API documentation

Build: [Strider](https://build.developer.rackspace.com/rackerlabs/docs-cloud-identity/)

## Purpose

This repository contains the source files that generate the following API documentation:

  • [Quickstart Guide](https://developer.rackspace.com/docs/cloud-identity/v2/#document-quickstart-guide)
  • [Cloud Identity Developer Guide](https://developer.rackspace.com/docs/cloud-identity/v2/)
  • [Cloud Identity Developer API Reference](https://developer.rackspace.com/docs/cloud-identity/v2/#api-reference)

### Support and feedback

We welcome feedback, comments, and bug reports. Follow the [contributor guidelines](CONTRIBUTING.md) to propose a source file change, or [submit a GitHub issue](https://github.com/rackerlabs/docs-cloud-identity/issues/new) to request an update or to provide feedback.

You can also contact the [Rackspace documentation team](mailto:devdoc@rackspace.com) directly for general issues or questions about the content.
